## 哈希表 
### hash table
#### where to use
+ 仿真映射关系
+ 防止重复
+ 缓存 / 记住数据，以免服务器再通过处理来生成它们
#### how to use
+ 良好的哈希函数
+ 较低的填装因子 （hash表的元素数 / 位置总数 > 0.7  ==> resize)
